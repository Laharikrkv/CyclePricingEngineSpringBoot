# Design Sensibility and UI

## 1. What is the most important thing on the configurator screen?
The most important element is the **main configuration panel** where the salesperson follows parts and select options. I kept this central and clear so they don’t have to search around the screen.

---

## 2. The salesperson uses this 20 times a day. What did you do to make repetitive use fast and easy?
Since the salesperson uses it many times a day, I focused on **reducing clicks** and keeping the layout straightforward. Common options are visible right away, and the flow is consistent so they can build muscle memory quickly. Even though I relied on AI assistance for the frontend, I made sure the design avoids unnecessary steps.

---

## 3. What happens when a part combination is invalid (e.g., tubeless tyre selected but no compatible rim)? How does your UI handle it?
The design ensures that **no invalid combination can be made**. For every essential part, one option must be selected, and until each required part is chosen, the configuration doesn’t add up. This prevents errors before they happen, so the salesperson only works with valid setups.

---

## 4. One thing you would improve if you had more time.
If I had more time, I would add **smarter suggestions** — for example, automatically highlighting compatible rims when a tubeless tyre is selected. That would make the salesperson even faster and reduce errors further.
